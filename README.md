# Higher Education Policy Analysis Using Quantitative Techniques — 2nd Edition

Welcome! This GitHub organization provides materials supporting Marvin Titus’s *Higher Education Policy Analysis Using Quantitative Techniques, 2nd Edition* (Springer).

---

## Repository Overview

This organization contains **four companion repositories**:

- **[Code](https://github.com/higher-ed-policy-analysis-2nd-edition/code):** Stata scripts for analyses; some R script translated from Stata
- **[Data](https://github.com/higher-ed-policy-analysis-2nd-edition/data):** Datasets in Stata, CSV, Excel formats
- **[Figures](https://github.com/higher-ed-policy-analysis-2nd-edition/figures):** Publication-quality graphics
- **[Tables](https://github.com/higher-ed-policy-analysis-2nd-edition/tables):** Supplementary statistical tables

**Navigation:**  
Each repo contains folders by chapter (`ch4`, `ch5`, ..., `ch12`). Start with the relevant chapter folder as you work through the book.

---

## Quick Start

**Students:**  
1. Read the book and select a chapter  
2. Download data from [Data](https://github.com/higher-ed-policy-analysis-2nd-edition/data)  
3. Run Stata scripts from [Code](https://github.com/higher-ed-policy-analysis-2nd-edition/code)  
4. Compare results with [Figures](https://github.com/higher-ed-policy-analysis-2nd-edition/figures) and [Tables](https://github.com/higher-ed-policy-analysis-2nd-edition/tables)

**Instructors:**  
- Integrate figures, assign code exercises, and adapt materials to your course

**Researchers:**  
- Replicate examples, adapt code and data, cite appropriately
